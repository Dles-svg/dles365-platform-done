# Miner Game Hosting Guide

The miner desktop app can now host games like Roblox for remote players to access. This allows gamers to connect to your equipment and play games installed on your computer.

## How It Works

1. **Install Games**: Install games on your computer (like Roblox, Fortnite, etc.)
2. **Add to Hosting**: Use the Hosted Games Manager to add games you want to share
3. **Test Launch**: Test that games launch correctly from the app
4. **Stream**: When broadcasting, gamers can connect and you can launch games for them

## Setting Up Roblox for Hosting

### Step 1: Install Roblox
- Download and install Roblox from https://www.roblox.com/download
- The typical installation path is: `C:\Users\[YourUsername]\AppData\Local\Roblox\Versions\RobloxPlayerLauncher.exe`

### Step 2: Access Hosted Games Manager
1. Open the Daylight ES365 Miner desktop app
2. Navigate to the **Streaming** page
3. Click **"Manage Hosted Games"** button

### Step 3: Add Roblox to Your Hosted Games
1. In the "Available Games" section, find **Roblox**
2. Click **"Add to Hosting"**
3. Roblox will appear in your "Hosted Games" section

### Step 4: Test the Game
1. Click the **"Test Launch"** button on the Roblox card
2. The game should launch on your computer
3. If it doesn't launch, the game path may need to be updated

## Custom Game Paths

If a game doesn't launch, you may need to update its executable path. Common locations:

- **Steam Games**: `C:\Program Files (x86)\Steam\steamapps\common\[GameName]\`
- **Epic Games**: `C:\Program Files\Epic Games\[GameName]\`
- **Riot Games**: `C:\Riot Games\[GameName]\`
- **Roblox**: `C:\Users\%USERNAME%\AppData\Local\Roblox\Versions\RobloxPlayerLauncher.exe`

## How Gamers Connect

1. Gamer browses your streaming session
2. They connect via your miner code
3. You can launch hosted games for them
4. They control the game remotely via WebRTC
5. You earn DL365 tokens based on rental time

## Revenue Split

When hosting games for gamers:
- **Platform Fee**: 10% goes to Daylight ES365
- **Your Earnings**: 90% of the rental fee
- Earnings are automatically calculated and deposited to your DL365 balance

## Tips for Best Performance

1. **Good Internet**: Upload speed of at least 10 Mbps recommended
2. **Game Installation**: Keep games updated and ready to launch
3. **System Resources**: Close unnecessary programs when hosting
4. **Test First**: Always test launch games before going live
5. **Set Availability**: Use the equipment availability schedule to manage when you're hosting

## Troubleshooting

**Game won't launch:**
- Verify the game is installed at the correct path
- Run the desktop app as administrator
- Check Windows permissions for the game folder

**Game launches but gamer can't control it:**
- Ensure remote input is enabled (shown in broadcaster view)
- Check firewall settings aren't blocking WebRTC
- Verify the gamer has a stable connection

**Performance issues:**
- Lower stream quality in broadcast settings
- Close background applications
- Check your upload bandwidth
- Upgrade your equipment tier for better rates
